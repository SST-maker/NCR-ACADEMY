# Registre de traçabilité pédagogique

Ce registre rassemble les éléments utiles au suivi qualité de la formation.

## Éléments suivis

Présence du stagiaire, modules consultés, activités réalisées, quiz effectués, score obtenu, certification pédagogique, tickets internes, observations formateur et documents remis.

## Utilisation dans la PWA

Dans cette version, le journal est stocké localement dans le navigateur et visible dans l’espace formateur. En production, ces données devront être enregistrées dans une base sécurisée avec gestion des droits d’accès.
