# N.C.R Academy PWA

Application web progressive de type LMS pour N.C.R Solutions.

## Version V2 contenu complet

Cette version contient les leçons, cas pratiques, critères de réussite, fiches mémo, lexiques, quiz et corrigés commentés pour :

- Communication interne et externe
- Outils numériques et bureautique
- Gestion et désamorçage des litiges clients
- Community Manager
- Acteur SST

## Comptes de démonstration

- Stagiaire : stagiaire@ncr.demo / ncr2026
- Formateur : formateur@ncr.demo / ncr2026

## Lancement

Le projet est statique. Il peut être servi avec n’importe quel serveur local, puis poussé directement sur GitHub Pages.

## Production

L’authentification, les documents et la traçabilité sont simulés localement pour permettre une base immédiatement testable. Pour une exploitation réelle, brancher Firebase Auth, Firestore et Storage ou un backend équivalent.

## Note SST

Les contenus SST sont reformulés pour la révision et l’entraînement. Ils ne remplacent pas l’enseignement pratique, les démonstrations ni l’évaluation officielle réalisée par un formateur habilité.
